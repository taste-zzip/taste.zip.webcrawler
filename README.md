# Youtube Shorts Web Crawler - taste.zip


## How to use (Mac, Linux 기준)

### Python 가상환경 venv 설치
`python3 -m venv venv`
### venv 실행
`source venv/bin/activate`

### pip packange 설치
`pip install -r requirements.txt`

### 검색 키워드 설정
`SEARCH_KEYWORD = '검색 키워드'.replace(' ', '+')`
